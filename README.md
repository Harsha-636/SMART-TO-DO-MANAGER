# Smart To-Do Manager

## Overview
Smart To-Do Manager is a Java-based task management application developed using Core Java concepts.

## Features
- Add Tasks
- Update Tasks
- Delete Tasks
- View Tasks
- File Storage
- Persistent Data Handling

## Technologies Used
- Java
- Collections Framework
- File Handling
- OOP Concepts

## Concepts Used
- ArrayList
- Classes & Objects
- Exception Handling
- CRUD Operations

## Future Improvements
- GUI Integration
- Database Support
- Login Authentication
